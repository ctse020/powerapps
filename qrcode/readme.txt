1) Create a new PowerApp project
https://docs.microsoft.com/en-us/powerapps/developer/component-framework/implementing-controls-using-typescript

2) In the command prompt go to the directory where the file package.json is situated and run this command: npm install qrcode --save

D:\Development\PowerApps\Controls\MyQRCode> npm install qrcode --save

3) Change the files "ControlManifest.Input.xml" and "index.ts"

4) In the command prompt go to the subfolder where the file "ControlManifest.Input.xml" is situated and run this command: npm start

D:\Development\PowerApps\Controls\MyQRCode\MyQRCode> npm start
